## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-ipblacklist/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Чёрный список IP-адресов»

Ограничение доступа пользователей к Панели управления GePanel и веб‑сайту путём блокировки подключений с указанного «черного» списка IP‑адресов.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Конфигурация»](https://github.com/rosgear/rg-be-config).

### Скриншот
<img src="https://github.com/rosgear/rg-be-config-ipblacklist/blob/main/assets/help/grid.png?raw=true">

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-config-ipblacklist
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-config-ipblacklist": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-config-ipblacklist/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

<a name="english"></a>
## <img src="https://raw.githubusercontent.com/rosgear/rg-be-config-ipblacklist/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «IP Blacklist»

Restrict user access to the GePanel Control Panel and website by blocking connections from a specified blacklist of IP addresses.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«Configuration» module](https://github.com/rosgear/rg-be-config).

### Screenshot
<img src="https://github.com/rosgear/rg-be-config-ipblacklist/blob/main/assets/help/grid.png?raw=true">

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-config-ipblacklist
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-config-ipblacklist": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-config-ipblacklist/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
